<h1 align="left">About</h1>

Minimal Notes is a rich-text note editor inspired by Notion and Obsidian.

The project was built to practice React state management and components, UI architecture and working with third-party documentation.

Access the <a href="https://pedroparisi" target="_blank">live preview</a> to check it out!
